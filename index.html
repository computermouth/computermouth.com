#!++ Crunchbang++

Early in 2015, the maintainer of the Linux distro #!(Crunchbang), announced that development was ceasing. A few months later, I launched an updated version. #!++ keeps exactly everything the same, with changes only implemented to stop breakage, because #! was perfect. Except now it's based on Debian Jessie.

Loclist is a project built using FCGI and SQL to provide an aggregation of map locations for companies with traveling employees.
